# 任务目标

实现列表页面数据展示：

基础功能：
 * 应用内嵌数据（10条）
 * 仅展示文字内容，简单排版（不要太丑）

进阶功能：
 * 应用内嵌数据（100条）
 * 展示文字和图片内容，排版美观

高阶功能：
 * 通过网络获取数据（100条）
 * 展示文字和图片内容，排版美观
 * 列表性能优化
